# OS Security - Windows

## 安全性更新

系統更新能避免多數的安全攻擊。**建議將自動更新開啟**

確認更新設定內容：

- Windows 7, Windows 8, Windows 8.1

    - 開啟 **我的電腦**
    - 點選 **開啟控制台**
    - 尋找並點選 **Windows Update**
    - 點選左列項目中的 **變更設定**
    - 確認 [重要更新] 中設定是否為 **自動安裝更新（建議選項）**
